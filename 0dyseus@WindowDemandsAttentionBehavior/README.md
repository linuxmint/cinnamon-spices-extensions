## Description

Extension based on the Gnome Shell extensions [Steal My Focus](https://github.com/v-dimitrov/gnome-shell-extension-stealmyfocus) and [Window Demands Attention Shortcut](https://github.com/awamper/window-demands-attention-shortcut). This extension adds two different behaviors for windows that demand attention. It force them to be activated or a keyboard shortcut can be assigned to activate windows on demand.
